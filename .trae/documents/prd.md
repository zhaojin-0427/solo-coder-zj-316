## 1. 产品概述

滴胶饰品层次排版与固化时间预览器——一款面向手作爱好者的纯前端工具，帮助用户在倒胶前预览滴胶饰品的材料层次布局、估算固化时间并排查潜在问题，避免实际制作中无法修改的遗憾。

- 解决滴胶饰品制作中"倒胶后难以调整层次"的核心痛点，让排版可视化、可试错
- 目标用户为手作滴胶饰品爱好者，提升制作成功率和创作效率

## 2. 核心功能

### 2.1 功能模块

1. **模具画布页**：模具选择（吊坠/发夹/戒指托/杯垫）、材料拖拽排版、层级调整、实时预览
2. **材料素材库面板**：干花、亮片、金箔、色粉四大类材料的素材选择与添加
3. **层级管理面板**：材料层级排序、透明度调节、厚度设定、锁定/删除
4. **固化时间预览区**：按层数与材料厚度估算总固化时间、分层数据展示
5. **智能提示系统**：遮挡警告、气泡风险、重心偏移提示
6. **方案管理**：保存多套排版方案到 localStorage、加载/删除/重命名方案
7. **导出功能**：导出排版图（PNG）、打印材料清单

### 2.2 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| 主工作台 | 模具选择器 | 选择吊坠/发夹/戒指托/杯垫四种模具，画布形状随之切换 |
| 主工作台 | 模具画布 | 显示模具轮廓，支持拖拽放置材料元素，支持选中/移动/缩放/旋转 |
| 主工作台 | 材料素材库 | 左侧面板，分类展示干花/亮片/金箔/色粉，点击或拖拽添加到画布 |
| 主工作台 | 层级面板 | 右侧面板，展示当前所有材料层列表，支持拖拽排序、透明度/厚度调整 |
| 主工作台 | 固化时间提示区 | 底部区域，显示预估固化时间、分层数据和智能提示 |
| 主工作台 | 方案管理栏 | 顶部工具栏，保存/加载/删除方案，导出排版图和材料清单 |

## 3. 核心流程

用户打开工具 → 选择模具类型 → 从素材库拖拽材料到画布 → 调整位置/大小/层级 → 系统实时计算固化时间并提示风险 → 保存排版方案 → 导出排版图或材料清单

```mermaid
    "固化时间计算" --> "风险提示"
    "风险提示" --> "保存方案"
    "保存方案" --> "导出排版图/材料清单"
    "调整位置与层级" --> "继续添加材料"
```

## 4. 用户界面设计

### 4.1 设计风格

- 主色调：琥珀色系（#D4A574）搭配深棕底色（#2C1810），呼应滴胶手作的温暖质感
- 辅助色：奶白（#FFF8F0）用于面板背景，金色（#C9A96E）用于强调交互
- 按钮风格：圆角胶囊按钮，带微阴影悬浮效果
- 字体：标题使用 Playfair Display，正文使用 Noto Sans SC
- 布局风格：三栏式（素材库 | 画布 | 层级面板），底部固化提示区
- 图标风格：lucide-react 线性图标，搭配手绘感装饰元素

### 4.2 页面设计概览

| 页面名称 | 模块名称 | UI 元素 |
|----------|----------|---------|
| 主工作台 | 模具选择器 | 4个模具卡片（带图标和名称），选中高亮，点击切换 |
| 主工作台 | 模具画布 | 居中画布区域，带模具轮廓线，材料元素可拖拽移动 |
| 主工作台 | 材料素材库 | 左侧250px宽面板，分类Tab（干花/亮片/金箔/色粉），素材网格 |
| 主工作台 | 层级面板 | 右侧280px宽面板，层级列表带拖拽排序，每层显示缩略图+名称+控件 |
| 主工作台 | 固化时间提示区 | 底部80px高区域，进度条+时间数字+提示标签 |
| 主工作台 | 方案管理栏 | 顶部60px工具栏，保存/加载/导出按钮，方案下拉列表 |

### 4.3 响应式

- 桌面优先设计，最小支持1280px宽度
- 三栏布局在小于1024px时，素材库和层级面板折叠为侧边抽屉
- 画布区域始终居中占满剩余空间
